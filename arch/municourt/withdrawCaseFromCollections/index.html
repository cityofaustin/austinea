Withdraw Case From Collections
Use Case Model

Scope: A case can be withdrawn from collections under any of the following four conditions:

  1. The Defendant has been arrested by an Officer
  2. The case has been dismissed by a Judge
  3. The case collection activity has exceeded its time limit
  4. The case has been withdrawn for other reasons:
    1. Case dismissed - re-filed
    2. Case disposed not guilty-transferred to Muni Court
    3. Defendant found not liable
    4. Case transferred out
      

This Use case also depends upon the following use cases:

  • Issue Warrants Use Case
  • Send Message To Collections Use Case

001 Issue Warrants
When the defendant fails to appear or comply with conditions on the case a warrant is issued.
002 Officer/processWarrant
The Officer receives and processes the warrant on the case by pursuing the Defendant, taking them into custody and delivering them to the Judge.

The case management system adds the case and a collection withdrawal flag to the collection send changes object.
004 Defendant/surrenderToCustody
The Defendant surrenders to custody.
005 Judge/deliverDefendantToJudge
The Officer delivers the Defendant to the Judge.
006 Manage Court Case/Send Message To Collections Use Case
The case management system creates a collections send file (coordinated with the collections vendor) for all cases added that includes the citation information, vehicle information (when applicable), personal identifiers and contact information, case starting balance, case balance, total calculated collection fee and status for any case that is being added or is already in the collections process. The entire data set is transferred to the collection vendor every time there is an update or change to the data. The collection vendor evaluates the entire file and compares the data values for each case to what they have on record. Any changes detected are used to add or update their records.
007 Judge/evaluateCase
The Judge evaluates the case for a dismissal condition.
008 Manage Court Case/dismissOrRejectCase
The judge reviews all elements of the case file, and selects one of two Case Rulings: "Dismiss" or "Reject" on the case selected from their work queue. The system attaches their signature,and adds the judge's Ruling to the case. If the judge rejects the case, they will annotate the case file with further instructions for the County Clerk. Dismissing the case indicates the case file is now completed, and the matter is closed. Withdrawn Dismissed disposition codes that qualify case selection for withdrawal form collections:
  • DEV Dismissed Insufficient Evidence
  • DIS Dismissed
  • DMA Dismiss - Juror
  • DMB Dismiss Govt Business - Meter Only
  • DMF Dismiss - News Media
  • DMJ Dismissed by Judge
  • DMS Dismissed - Sold Vehicle
  • DMZ Dismissed - Capitol Complex
  • DM5 Dismiss Motion by State & Order by Court
  • DNN Dismissed Officer Not Notified
  • DOF Dismissed Officer Not Present
  • DRE Dismissed - Re-filed
  • DSL Dismissed - State Legislature
  • DML Dismiss - Humanitarian
  • DMO Dismiss - Ticket Error Fatal Error
  • DMQ Dismiss - Officer Request Retract Ticket
  • DM2 No Complaint Filed Within 2 Years of Violation
  • NLA Case Closed Defendant Deceased
009 Manage Court Case/monitorCollectionsTimeLimit
The case management system monitors the elapsed time that the case has been in collections without any activity by the collection service.
010 Manage Court Case/processCompletionForm
When the case is withdrawn from collections for other reasons: The Court Clerk processes the submitted mandated program completion form from the Defendant.
Codes indicating a withdrawl for other reasons:
  • DRE   Dismissed - Refiled
  • NGT   Not Guilty-Transferred to Muni Court
  • NLB   Not Liable
  • TO    Transferred Out
  • CAC   Completed Alcohol Awareness Class
  • CDS   Completed Driving Safety Course
010 Judge/evaluateCase
If the Judge makes a collection withdrawal qualifying disposition on the case: The Judge evaluates the case for a dismissal condition.
011 Judge/makeDisposition
The Judge makes a disposition on the case.
012 Manage Court Case/addActionToCase
The case management system adds an action, specified by an action code, to the case (referenced by the case number) to create the action record. If a DSC completion paperwork is being submitted the action reflects the sentence completion.
