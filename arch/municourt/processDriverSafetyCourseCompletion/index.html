Process Driver Safety Course Completion
Use Case Model

Scope: The Court Clerk processes the submitted mandated DSC program completion form from the Defendant: The defendant presents the mandated DSC program completion form to the Court Clerk. The Court Clerk takes the form and processes it by scanning it onto the case and creating a case action recording the completion of service.

Process Driver Safety Course Completion
Sequence Diagram

Req#Type/Interface - Requirement DescriptionPerformance
001 Court Clerk/processCompletionForm
The Court Clerk processes the submitted mandated program completion form from the Defendant.
002 Manage Court Case/processCompletedProgramForm
The case management system processes the completed program form noting the program type completed, the document submitter and the date the program was completed.
003 Manage Court Case/addActionToCase
The case management system adds an action, specified by an action code, to the case (referenced by the case number) to create the action record. If a DSC completion paperwork is being submitted the action reflects the sentence completion.
004 Manage Court Case/attachDocumentToCase
The case management system takes the document which has been uploaded and attaches it to the case file. The document can be any document when the use case is "Accept Case File document."
If it is a case generated from a CARL citation the documents are:
  1. The two images of the violating vehicle - position in the intersection and zoomed in license plate.
  2. The citation
If a case is being generated from a parking citation, the document is the citation.
If it is a criminal parking case the documents are:
  1. The affidavit
  2. The placard
  3. The scanned image of the customer's driver license
If a cash bond is being posted the document is the bond.
If a scheduled time payment plan is being established or modified the document is the STPP.
If an appeal is being processed the documents are:
  1. Order ofTransfer (2 copies)
  2. Certificate of Transfer
  3. Approval of Trial Judge
  4. Letter to Defendant
  5. Letter to Prosecutor
  6. Transcription file
  7. The appeal document
If a late representation request is being filed the documents are: Attorney's letter of representation
  1. The motion to proceed
  2. Appearance motion
  3. Emergency order or motion
If a customer has completed a Driver Safety Course the document is the completion certificate.
If a probable cause affidavit is being filed, the document is the affidavit
If an appeal is being initiated the documents are:
  1. The motion to proceed document
  2. The appeal bond
If a general or Youth Services complaint is being generated, the document is the complaint.
005 Manage Court Case/scanCaseDocument
The Court Clerk scans the document onto the case. The case management system attaches the document to the case.
