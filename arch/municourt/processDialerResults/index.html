Process Dialer Results
Use Case Model

Scope: Following the processing of a phone dialer campaign, the phone dialer service compiles the results for all calls placed and generates a return file that is processed by the case management system. The CMS Scheduler initiates a transfer of the file to the case management system. The action for each call is stored on each case. If a Defendant was unsuccessfully contacted but the call was not in error, then the Defendant can be recontacted in a subsequent campaign as qualifying criteria allow.

003 CMS Scheduler/initiateDialerResultProcessing
The CMS Scheduler initiates a process (for each notification campaign a different times during the day) to transfer (ftp) the dialer download file to a location where the case management system can access, read and process the campaign results.

006 Manage Court Case/processDialerDownload
Until all dialer campaign results have been processed: The case management system reads the case number and dialer service results and extracts the action code that will be put on the case. If the action code is final then the case will be excluded form future notification campaigns. Action codes include:
 
DDOKBSY = Dialer Call Busy
DDOKERR = Dialer Call Error
DDOKFAX = Dialer Call Answered by Fax
DDOKHUM = Dialer Call Answered
DDOKMOD = Dialer Call Answered by Modem
DDOKNOA = Dialer Call No Answer
DDOKNOT = Remove Initial Code
DDOKOPR = Dialer Call Phone Number No Longer in Service
DDOKVOI = Dialer Call Left Voice Message
DIALDOK = Pre Docket Dialer Call
DIALERR = Dialer call attempt resulted in an error
DIALFAX = Dialer call answered by fax
DIALINT = Pre Initial Appearance Dialer Call
DIALMOD = Dialer call answered by modem
DIALNOA = Dialer call not answered
DIALOPR = Dialer call - phone number no longer in service.
DIALPAS = Past Due Dialer Call
DIALSTP = Past Due STPP Dialer Call
DIALVOI = Dialer call left voice message
DIALWAR = Warrant Dialer Call
DIALWRP = Warrant Amnesty Prk Dialer Call
DIALWRT = Post Warrant dialer call
DIALWRU = Warrant Amnesty Traff Dialer Call
007 Manage Court Case/determineLegalOptions
The case management system adds an action, specified by an action code, to the case (referenced by the case number) to create the action record.
If a DSC completion paperwork is being submitted the action reflects the sentence completion.
If the action records a missed appearance with the prosecutor then the action codes include: TRAD, CMAD or ENAD
If the action records a missed trial then the action codes include: JTRI, BTRI
