Accounting For Community Service Time
Use Case

Use Case Scope

The Defendant submits proof of Community Service hours. The Clerk locates the Community Service Plan, and applies the hours reported to the Community Service Plan. The Clerk attaches the Community Service document to the Community Service Plan. The upload creates a unique reference number for tracking the "Receipt" document. The System offers the clerk the option to either print or email the Receipt to the Defendant/Attorney/etc., and attaches a Receipt to the Community Service Plan and associated credited case, for future printing. The System updates each case affected with a case specific credit for the associated Community Service and records the associated reference document number of the Scanned Community Service hours. Both Dollar and Hour totals are reflected at the Case level. The system adjusts the balances against the plan, and generates a receipt for the Defendant. The Community Service Plan update is added to the Clerk queue for later confirmation. The Clerk contacts the Community Service Provider to verify the hours submitted.

Accept Case File Document Interface Descriptions

Type/InterfaceDescription
Manage Court Case/
locatePID		 The User is presented a list of fields to provide information for the system to locate the Person ID (PID) for a defendant. The fields used for the PID search include, but are not limited to, Partial First name, Partial Last Name, First Name, Last Name, Address, Driver's License Number, Plate Number, Date of Birth, Telephone number. The System conducts a search and provides a list of matching Defendants from which the User can choose.
Manage Court Case/
locateCommunityServicePlanByDefendant		 The User specifies the Personal ID (PID) for the defendant, or selects the name of a defendant which provides the PID to be used for subsequent actions. The system returns a designator indicating the number of Community Service Case plans the Defendant has. If at least one exists, the system returns a list of plans established. The list of Plans also includes a list of the associated cases for each plan. Additionally the system returns a list of ALL cases associate with the PID not currently assigned to a Community Service Plan.
Manage Court Case/
applyHoursToCommunityServicePlan		 The Clerk selects the Community Plan listed which is associated with the case whose hours are to be credited. The System prompts the Clerk to provide documentation/proof to be scanned. Once the document is scanned, the Clerk verifies the scan is acceptable,and is offered the opportunity to rescan the document should the image not be satisfactory. Once acceptable, the system generates a Receipt which should be stored and printable. The Receipt is locatable and printable by both the case and the community service plan to which the case belongs.
Manage Court Case/
specifyReceiptOptions		 The System provides a selection for managing Receipts associated with a Case or Community Service Plan. The Clerk can either print or email a receipt.
Manage Court Case/
sendEmail		 The Clerk has requested that a Receipt for Community Service Hours be emailed to an address. The Clerk specifies the email Address destination, or references a person associated with the case (Defendant or Attorney) whose email is registered and verified with the system. The Clerk is given the opportunity to add multiple recipients if desired.
Manage Court Case/
printReceipt		 The Clerk has requested that a Receipt for Community Service Hours be Printed. The Clerk specifies the printer at which the Community Service Receipt will be printed. The Clerk is given the opportunity to print multiple copies if desired.
Manage Court Case/
addCommunityPlanUpdateToClerkQueue		 A Community Service Plan has been updated in some fashion (hours applied, new cases added to the plan, cases removed from the plan, or plan deleted). The Community Service plan is added to the Clerk Queue for referential integrity checks; the Clerk will reach out to each community service provider the Defendant has listed, to verify the hours worked (credited) at a later date.
