Modify PID
Use Case

Use Case Scope

The Clerk conducts a PID search with any or all of the following information: Partial First name, Partial Last Name, First Name, Last Name, Email Address, Alias, Address, Case no. , Driver's License Number, Plate Number, Date of Birth, Telephone number and/or PID id. If there is ambiguity, The system presents a list of matches for the search, and the Clerk specifies the matching person. The Clerk makes the desired changes to the PID record, and commits the changes. The Clerk can add Cases to the PID, Merge another PID (consolidating two PIDs into one), remove Cases from a PID, or even Delete a PID as necessary. The System stores a historical snapshot of the information prior to change, and an audit log is written to indicate what was changed on the PID, who changed it, and the date on which it was done. The System creates an event (Action) indicating the appropriate change to all open cases. Any query against an open Case will show this action listed. For example, if an address is changed on the PID, each case associated with the same Defendant (PID) will also reflect the action of an address change.

Manage Court Case/
locatePID		 The User is presented a list of fields to provide information for the system to locate the Person ID (PID) for a defendant. The fields used for the PID search include, but are not limited to, Partial First name, Partial Last Name, First Name, Last Name, Address, Driver's License Number, Plate Number, Date of Birth, Telephone number. The System conducts a search and provides a list of matching Defendants from which the User can choose.
Manage Court Case/
modifyPID		 The Clerk makes the desired changes to the PID record, and commits the changes. The System stores a historical snapshot of the information prior to change, and an audit log is written to indicate what was changed on the PID, who changed it, and the date on which it was done. The System reflects the changes in all open cases associated with the PID. Any query against a Case will show the changes listed. For example, if an address is changed on the PID, each case associated with the same Defendant (PID) will also reflect an address change has occurred. The old address will still be recoverable in the historical record associated with the PID.
