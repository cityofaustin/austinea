Inspect Lodging Business
Use Case Model

Scope: If an Inspection is required after a LodgingLicense was issued, the Austin Code Officer performs a follow-up Inspection of the lodging business and takes Photographs. The Austin Code Officer updates the LodgingLicense in the Case Management System with the results of their Inspection. The Austin Code Officer performs two follow-up Inspections on the lodging business, and provides the lodging business with time to correct Violations. If Violations are not resolved after the second Inspection, the Austin Code Officer begins the Inspect Property Complaint use case. If a mobile home park's LodgingLicense has expired, the Austin Code Program Specialist requests the Austin Code Officer follow up with the LodgingLicense's Owner. The Austin Code Officer follows up with the LodgingLicense Owner to help resolve the expired LodgingLicense. If Violations are identified, the Austin Code Officer begins the Inspect Property Complaint use case.

Sequence Diagram

Req#Type/Interface - Requirement DescriptionPerformance
001 Austin Code Officer/performFollowUpInspection
Follow-up Inspection for Lodging License: The Austin Code Officer checks for GFCI outlets, smoke detectors (set off and sound), trip hazards, cracked windows, broken windows, weather stripping, temperature of hot water, pest infestation, secured balcony railing, overhead light inspection outside, loose shower spouts, secured toilet, faucet handles, locks on windows (if three feet up). Ten percent of the total rooms are inspected. Checklist: Structure Maintenance Report. Follow-up inspection for Waste Hauler: A WasteHaulerApplication is turned in, and the Austin Code Officer works with the Austin Code Program Specialist to ensure the WasteHaulerApplication is complete. If Illegal Dumping, the Austin Code Officer ensures property is abated and takes photographs.
002 Case Management System/updateLodgingLicense
The Austin Code Officer updates the LodgingLicense in the Case Management System with results of their Inspection and Photographs.
003 Austin Code Officer/followUpOnLodgingLicenseWithOwner
The Austin Code Program Specialist requests the Austin Code Officer follow up with the LodgingLicense Owner to discuss their expired LodgingLicense. The Austin Code Officer works with the LodgingLicense Owner to correct Violations, complete the LodgingLicenseApplication, or collect Check for LodgingLicense.
