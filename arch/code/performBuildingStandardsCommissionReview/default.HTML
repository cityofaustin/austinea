Perform Building Standards Commission Review
Use Case Model

Scope: The Austin Code Legal Coordinator creates the BSCAgenda for the Building and Standards Commission (BSC) meeting. The Austin Code Legal Coordinator emails the BSCAgenda to the City Clerk. At least 72 hours in advance of the BSC meeting, the BSCAgenda is uploaded to the City of Austin's website by the City Clerk. The Chain of Command sends Notification of prospective BSC cases to the relevant Austin Code Officers and their Chain of Command. Notification is typically sent the first business day of the month. The Austin Code Legal Coordinator develops a BSCPresentation based on the BSCAgenda. Two weeks before the BSC meeting, the Austin Code Legal Coordinator conducts a pre-BSC meeting to review the cases with the Austin Code Officer and Chain of Command. Ten calendar days before the BSC meeting, the Austin Code Legal Coordinator sends Notification to the legally interested parties of the BSC cases (Owner, Tenant, etc.). The BSC meets and evaluates the cases using the BSCPackets' information. The BSC determines whether to issue an Order, accept and modify an Order, or defer a ruling (to allow Property to come into compliance, future BSC review, additional information required, Property Owner unavailable, etc.). The BSC Meeting is recorded for future transcription of meeting minutes. If the BSC accepts the recommendation for the case, or accepts with modification, the Austin Code Legal Coordinator creates an Order and has the BSC sign the Order. The LegalCase is updated in the Case Management System. The Legal Coordinator then sends an issued Order Notification to the County Clerk, City Clerk and Owner/Responsible Party. The Austin Code Legal Coordinator uses a Contractor to transcribe the BSCMinutes from the BSCMeetingRecording. The BSCMinutes are uploaded to a network location and attached to the relevant LegalCases in the Case Management System. The Owner/Responsible Party is provided a time period to comply with the Order. If the Owner doesn't display initiative to comply for a demolition Order, Austin Code begins the demolition process. The Austin Code Legal Coordinator requests Public Works coordinate a demolition bid with Contractors. The Legal Coordinator then notifies utilities to disconnect service and the demolition occurs. After the demolition, the Austin Code Officer performs an Inspection and updates the CodeViolationCase. The Pay Contractor and Bill Owner use case is invoked. If the BSC defers their decision, then nothing occurs until the BSC reviews the case again. The LegalCase is closed when the Property is in compliance and all penalties/fines are paid.

Perform Building Standards Commission Review
Sequence Diagram

Req#Type/Interface - Requirement DescriptionPerformance
001 Austin Code Legal Coordinator/updateBSCAgenda
The Austin Code Legal Coordinator creates the agenda for the BSC meeting. This is a roster of cases to be presented that month. The BSCAgenda is emailed to the members of the BSC and the City Clerk. The City Clerk uploads the BSCAgenda 72 hours before the BSC meeting.
002 Chain of Command/sendPreBSCMeetingNotification
The Austin Code Legal Chain of Command sends an emailed Notification of prospective BSC cases to the relevant Austin Code Officer and their Chain of Command. Notification is typically sent the first business day of the month.
003 Austin Code Legal Coordinator/developBSCPresentation
The Austin Code Legal Coordinator prepares the BSCPresentation that includes all documentation (case history, postings, pictures, recommended order for the property, maps, TCAD/WCAD information, registered agents information [if applicable], NoticeOfViolation, notice hearings, postings of NOV and hearing, and newspaper posting).
004 Austin Code Legal Coordinator/conductPreBSCMeeting
The Austin Code Legal Coordinator conducts the pre-BSC meeting to review all BSC cases for the upcoming month and walks through cases with the Austin Code Officer and Chain of Command. Two weeks in advance of the BSC meeting, a pre-BSC meeting is conducted, and is held at the same location and time each month.
005 Austin Code Legal Coordinator/sendBSCNoticeToLegallyInterestedParties
The Austin Code Legal Coordinator sends BSCNotice to legally interested parties (Owner, Tenant, etc.) for the upcoming BSC cases.
006 Building and Standard Commission/evaluateBSCCase
The Austin Code Officer, Austin Code Legal Coordinator, Austin Code Legal Admin and Chain of Command (Code Officer and Austin Code Legal) present the case to the Building and Standards Commission. Additionally, witnesses may provide information during BSC hearing. The Building and Standards Commission determines whether to issue an Order, accept and modify an Order, or defer a ruling (to allow Property to come into compliance, future BSC review, additional information required, property owner unavailable, etc.). The BSC Meeting is recorded for future transcription of meeting minutes.
007 Building and Standard Commission/alterBSCCaseRecommendationAsAppropriate
The Building and Standards Commission modifies the BSC case recommendations during the meeting.
008 Austin Code Legal Coordinator/createOrder
The Austin Code Legal Coordinator creates the Order based on the BSC disposition. The Order includes timeframe, penalties, whether the Order is for repairs or demolition, details of repairs, permits pulled, engineer's report, plan to resolve, milestones, additional inspections and final inspections. A CoverLetter identifies meeting date, what decision was made and that Order will be filed at County Clerk's office. The Austin Code Legal Coordinator has the Order notarized and signed by the BSC Chair.
009 Case Management System/updateLegalCase
The Austin Code Legal Coordinator accesses the system and updates the relevant case with updates or follow-ups from the BSC hearing (BSCDetermination, Order, etc.). The Austin Code Legal Coordinator updates the LegalCase to include the CauseNumber and final ruling of the case. The Austin Code Administrative Hearing Coordinator updates the LegalCase with the date of the hearing, outcome of hearing, penalty, if ComplianceHearingNeeded, if penalty was paid, if penalty needs to be billed, and if the Violator came into compliance.
010 Austin Code Legal Coordinator/sendIssuedOrderNotification
The Austin Code Legal Coordinator drafts a letter indicating the BCS Order, and files it with the County Clerk's office (Travis, Williamson or Hays) and the City Clerk's office. The Order is sent to the Owner/Representative/Responsible Party by certified mail, regular mail and posted at the Property by an Austin Code Officer or Legal Officer.
011 Contractor/transcribeBSCMinutes
The Austin Code Legal Coordinator requests a Contractor to transcribe the BSCMeetingRecording into BSCMeetingMinutes. A vendor contract exists already for meeting transcription.
012 Austin Code Legal Coordinator/uploadBSCMeetingMinutes
The Austin Code Legal Coordinator reviews the minutes and archives them on a network location. BSCMeetingMinutes are uploaded to the City of Austin's City Clerk's Office's website after the next BSC meeting. Additionally, the BSCMeetingMinutes are attached to the LegalCases in the Case Management System.
013 Public Works/coordinateDemolitionBid
The Austin Code Legal Coordinator requests Public Works to coordinate a bid with Contractors to demolish the Property. The Austin Code Legal Coordinator provides the Order.
014 Austin Code Legal Coordinator/notifyUtilitiesToDisconnectService
The Austin Code Legal Coordinator notifies Utilities to disconnect service to the Property based on the Order.
015 Austin Code Officer/performInspection
If the violation concerns hazardous conditions, the Austin Code Officer applies Personal Protective Equipment (PPE). The Austin Code Officer inspects the Property. The Austin Code Officer inspects from the exterior (initially from the right of way) to the interior (if the interior is accessible and permission has been granted by legally interested party or someone in Care/Control of Property). The Austin Code Officer takes photographs and measurements as needed (hot water temperature, air quality, interior temperature, light meter, ground-fault circuit interrupter, gas meter, carbon monoxide, smoke detector, fences, grass height, driveway and walkways, checks impervious cover, etc.). As needed, the Austin Code Officer performs research to identify the Property Owner, offense, ViolationType, OwnerInformation, and relevant PropertyInformation (GIS system, ARR Site, annexation information, AMANDA permit history, site plan, TCAD/WCAD for OwnerHistory and DeedHistory, CC&B, ZoningHistory, CodeOfOrdinances, etc.). The Austin Code Officer interviews relevant parties. For Waste Hauler Inspection: The Austin Code Officer makes contact with driver, visually inspects vehicle for decals on door, asks driver about awareness requirement for waste hauler license, ensures driver is operating within City of Austin (outside of city is exempt), double checks the Waste Hauler license, checks for safety-equipment violations, driver's license of driver, and takes photos of truck (including both doors and front license plate). If a short-term rental complaint, the Austin Code Officer responds more quickly than other complaints. On Fridays and Saturdays, the Austin Code Officer performs inspection during day, and from 6:00 p.m. - 2:00 a.m. On Sundays, inspections occur from 8:00 a.m. - 12:00 p.m.
016 Case Management System/closeCodeViolationCase
Once all Violations have been resolved, the CodeViolationCase is closed by a user or the system.
017 Case Management System/closeLegalCase
The user uses the system to close the LegalCase. LegalCases are only closed once the Penalty has been paid for the LegalCase, or if the case was dismissed.
