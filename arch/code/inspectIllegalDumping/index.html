Inspect Illegal Dumping
Use Case Model

Scope: If Surveillance Cameras need to be installed to monitor illegal dumping, the Austin Code Officer configures the Surveillance Camera's RecordingSettings and then installs the Surveillance Camera at the desired location. The Austin Code Officer monitors daily the Surveillance Cameras for illegal dumping, which entails visiting the Surveillance Cameras and retrieving photographs. The Austin Code Officer then resets the Surveillance Camera and researches the photographs for illegal dumping. If the Offender is identifiable through the photographs, the Austin Code Officer begins the Issue Violation Citation use case, and orders the Offender to abate the property. The Austin Code Officer performs a follow up inspection to ensure the property is abated and takes additional photographs. The Austin Code Officer accesses the Case Management System and creates an IllegalDumpingCase. The IllegalDumpingCase is closed when the associated LegalCase, if created, is closed, or if no LegalCase is created due to no Offender being identified.

Inspect Illegal Dumping
Sequence Diagram

Req#Type/Interface - Requirement DescriptionPerformance
001 Surveillance Camera/configureRecordingSettings
The Austin Code Officer configures the Surveillance Camera's settings, including the interval to record, number of photographs to take, how fast to take photographs, and the date and time.
002 Austin Code Officer/installSurveillanceCamera
The Austin Code Officer installs the Surveillance Camera, which are installed in a variety of locations. The Austin Code Officer monitors and performs maintenance on the Surveillance Cameras as necessary, including replacing batteries, charging batteries, etc.
003 Surveillance Camera/providePhotographs
The Austin Code Officer downloads photographs from the Surveillance Cameras. Some Surveillance Cameras require a manual download from hardware; other Surveillance Cameras have a wireless download (pole cameras, surveillance cameras, flash cams, etc.). The Austin Code Officer deletes the photographs to free up memory, and resets the Surveillance Cameras ("rearming").
004 Austin Code Officer/researchIllegalDumping
The Austin Code Officer reviews the photographs from the Surveillance Cameras and performs an on-site inspection of the location for Offender identifiable information. The Austin Code Officer reviews the photographs searching for vehicle information, including vehicle type, color and license plate.
005 Offender/abateProperty
The Austin Code Officer requests the Offender to abate the property.
006 Austin Code Officer/performFollowUpInspection
Follow-up Inspection for Lodging License: The Austin Code Officer checks for GFCI outlets, smoke detectors (set off and sound), trip hazards, cracked windows, broken windows, weather stripping, temperature of hot water, pest infestation, secured balcony railing, overhead light inspection outside, loose shower spouts, secured toilet, faucet handles, locks on windows (if three feet up). Ten percent of the total rooms are inspected. Checklist: Structure Maintenance Report. Follow-up inspection for Waste Hauler: A WasteHaulerApplication is turned in, and the Austin Code Officer works with the Austin Code Program Specialist to ensure the WasteHaulerApplication is complete. If Illegal Dumping, the Austin Code Officer ensures property is abated and takes photographs.
007 Case Management System/createIllegalDumpingCase
The Austin Code Officer uses the Case Management System to create a IllegalDumpingCase. The Austin Code Officer provides the Location, Photographs (Surveillance Camera, Austin Code Officer photos taken at site, clean up), Inspection, Synopsis of dumped items, Offender (if available), information on Property Abatement, and Citation (if Offender identified). If Offender identified the IllegalDumpingCase remains open until the LegalCase is completed.
008 Case Management System/closeIllegalDumpingCase
The Austin Code Officer requests the Case Management System close the IllegalDumpingCase. Cases are closed if no Offender is identified and property is abated; or, if the Offender is identified, after the LegalCase is closed.
