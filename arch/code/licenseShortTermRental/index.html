License Short-Term Rental
Use Case Model

Scope: A Short-Term Rental Applicant (Citizen or an Authorized Agent) completes a ShortTermRentalApplication. The Short-Term Rental Applicant either submits their ShortTermRentalApplication by mail or in person. If the ShortTermRentalApplication is submitted by mail, Austin Code Finance processes the ShortTermRentalApplication, initiates the Process Payment use case, and forwards the ShortTermRentalApplication to the Austin Code Program Specialist for further processing. If the ShortTermRentalApplication is submitted in person, the Austin Code Program Specialist reviews the ShortTermRentalApplication for completeness, and if incomplete, the Austin Code Program Specialist returns the application to the Short-Term Rental Applicant. The Austin Code Program Specialist processes the check for the ShortTermRentalApplication, and returns a receipt to the Short-Term Rental Applicant. The Austin Code Program Specialist then requests Austin Code Finance to start the Process Payment use case. After the ShortTermRentalApplication has been accepted, the Austin Code Program Specialist uses TCAD to verify Owner information, as well as verifies property information in the ARR Profile Viewer, and looks up property history in the Case Management System. If the ShortTermRentalApplication is for a type-2 ShortTermRental, the Austin Code Program Specialist uses the Tract Access Database to verify ShortTermRentalLicense availability. If the ShortTermRentalApplication is to renew a ShortTermRentalLicense, the Austin Code Program Specialist looks up the ShortTermRentalLicense. Then the Austin Code Program Specialist determines the ShortTermRentalApplication status: incomplete, CodeComplaints exist for the property, tract is percentage capped, rejected, or complete and accepted. If the ShortTermRentalApplication is incomplete, the Austin Code Program Specialist requests the Short-Term Rental Applicant complete the ShortTermRentalApplication. If no progress is made by the Short-Term Rental Applicant to complete the ShortTermRentalApplication, then the use case ends and must be restarted. If the Property has CodeComplaints, the Austin Code Program Specialist sends the ShortTermRentalApplication to the Division Manager for review. The Division Manager then decides whether to accept the ShortTermRentalApplication or if the STR Review Committee needs to make a decision. If the STR Review Committee is required, the Austin Code Program Specialist sends them the ShortTermRentalApplication, and the STR Review Committee determines whether to accept or reject the ShortTermRentalApplication. If the tract is percentage capped, the Austin Code Program Specialist notifies the Short-Term Rental Applicant and asks if they would like to be added to the waiting list. If the Short-Term Rental Applicant chooses to be placed on the waiting list, they sign the WaitingListAgreement. If a check was previously cashed for the ShortTermRentalApplication, the Austin Code Program Specialist requests Austin Code Finance issue a refund, and Austin Code Finance starts the Pay Invoice use case to issue the refund. If the ShortTermRentalApplication is complete and accepted, the Austin Code Program Specialist performs a final review of the ShortTermRentalApplication, looks up businesses associated with the property (if any businesses identified on the ShortTermRentalApplication), and then creates the ShortTermRentalLicense in the Case Management System. The Austin Code Program Specialist requests the Division Manager activate the ShortTermRentalLicense. The Division Manager activates the ShortTermRentalLicense in the Case Management System. After the ShortTermRentalLicense is activated, the Austin Code Program Specialist mails the ShortTermRentalLicense to the Short-Term Rental Applicant. The Austin Code Program Specialist updates the ShortTermRentalLicense in the Case Management System with the ShortTermRentalApplication and other documents. At the start of the month, the Austin Code Program Specialist sends a report of new ShortTermRentalLicenses to the City Controller.

License Short-Term Rental
Sequence Diagram

Req#Type/Interface - Requirement DescriptionPerformance
001 Short-Term Rental Applicant/completeShortTermRentalApplication
The Short-Term Rental Applicant, either an Owner or an Authorized Agent, completes a ShortTermRentalApplication indicating date opens, number bedrooms, number people, contact person, estimated charge per night, Owner, insurance information, and signatures (must be original signatures). Additionally, the Short-Term Rental Applicant provides supporting documentation such as declarations' page for homeowner's insurance, certificate of occupancy, or third-party inspection (has to be completed using Austin Code's Third-Party Inspection Form). If an Authorized Agent, a notarized authorization from Owner for Authorized Agent to submit application is required. There are three types of ShortTermRentalApplications: Type 1: Must have homestead exemption included, Type 2: Must have homestead exemption removed, and Type 3: Official Property description including number of buildings on property and units per building. If a Corporation owns Property, FormationDocuments with original signatures from managing partners, along with notarized authorization from Owner. If renewing a ShortTermRentalApplication, requires updated insurance, most recent city tax report (hotel occupancy tax), front and back of cancelled check, and a check for the renewal fee. If the Short-Term Rental Application is incomplete, the Austin Code Program Specialist contacts the Short-Term Rental Applicant to submit the required additional documentation.
002 Austin Code Finance/processShortTermRentalApplication
The Short-Term Rental Applicant submits the ShortTermRentalApplication by mail to Austin Code Finance. Austin Code Finance cashes the check from the Application. Type-2 applications are not processed by Austin Code Finance. Austin Code Finance returns the application to the Short-Term Rental Applicant.
003 Austin Code Program Specialist/processShortTermRentalApplication
If ShortTermRentalApplication was submitted by mail, Austin Code Finance submits the ShortTermRentalApplication to the Austin Code Program Specialist (in person). If a mailed-in application, the Austin Code Program Specialist contacts the Short-Term Rental Applicant for any missing supporting documentation. The Austin Code Program Specialist processes the ShortTermRentalApplication. If a check was provided in person, the Austin Code Program Specialist processes the payment, returns a receipt, and requests Austin Code Finance record the Payment.
004 Austin Code Program Specialist/processPayment
The Austin Code Program Specialist receives a check and deposits it with Austin Code Finance. The Austin Code Program Specialist returns a receipt to the Short-Term Rental Applicant for the check.
005 TCAD/verifyOwnerInformation
The Austin Code Program Specialist researches Owner information in TCAD, such as who's Owner, type of building (duplex, condo, fourplex, garage apartments), legal dwelling unit, and homestead exemption.
006 ARR Profile Viewer/verifyPropertyInformation
The Austin Code Program Specialist uses the ARR Profile Viewer to verify property information, such as zoning, census tract, jurisdiction (ETJ or full purpose), etc.
007 Case Management System/lookupPropertyHistory
The Austin Code Program Specialist performs research in the Case Management System: Owner, property's CodeComplaints, violations, and trade permit issues. If a Type-3 application, the Austin Code Program Specialist checks for the number of Short-Term Rentals for the property.
008 Tract Access Database/verifyShortTermRentalAvailability
The Austin Code Program Specialist researches if the Census Tract in which the property is located hasn't reached the cap (only applies to Type-2 ShortTermRentals).
009 Case Management System/lookupShortTermRentalLicense
The Austin Code Program Specialist looks up the ShortTermRental in the Case Management System, including expiration date and operating license number.
010 Austin Code Program Specialist/determineShortTermRentalApplicationStatus
The Austin Code Program Specialist determines the ShortTermRentalApplicationStatus: If the application is missing documentation, or trade permit issues, the Short-Term Rental Applicant is advised of the issues to resolve and must provide documentation before review commences. If needed, the Austin Code Program Specialist provides copies of expired trade permits from the Case Management System. If the property has CodeComplaints, the Application is sent to the Short-Term Rental Review Committee. If the property has violations, the violations must be resolved before license is issued. If violation type is work without permit or too many in occupancy, then the application is forwarded to the Division Manager for review. The Division Manager either approves, approves conditionally, or forwards to the STR Review Committee.
011 Division Manager/reviewShortTermRentalApplication
The Austin Code Program Specialist requests the Division Manager review the ShortTermRentalApplication. The Division Manager reviews the application, property history of CodeComplaintCases, and violations, and then determines whether to accept the application, accept conditionally, or forward to the STR Review Committee. The Division Manager notifies the Austin Code Program Specialist of their decision.
012 STR Review Committee/reviewShortTermRentalApplication
The STR Review Committee reviews the ShortTermRentalApplication and determines whether to accept, accept conditionally, or reject the ShortTermRentalApplication.
013 Short-Term Rental Applicant/signWaitingListAgreement
The Austin Code Program Specialist contacts the Short-Term Rental Applicant and provides a WaitingListAgreement for the applicant to sign, if they choose to be placed on the WaitingList for ShortTermRentals. The applicant may sign or choose to not be placed on the WaitingList.
014 Austin Code Program Specialist/addShortTermRentalApplicantToWaitingList
The Austin Code Program Specialist adds the Short-Term Rental Applicant to the ShortTermRentalWaitingList.
015 Austin Code Finance/issueRefund
The Austin Code Program Specialist requests Austin Code Finance to issue a refund for a cashed check.
016 Austin Code Program Specialist/sendRejectionNotification
The Austin Code Program Specialist sends a Notice of Denial Notification via regular mail to the Short-Term Rental Applicant. The Short-Term Rental Applicant has the option to appeal to the Building and Standard Commission if desired.
017 Austin Code Program Specialist/reviewShortTermRentalApplication
The Austin Code Program Specialist reviews the ShortTermRentalApplication. The Austin Code Program Specialist reviews all documentation for the application, including that the signatures are original, and that the licenses are not capped within that Census Tract. This is a final check before the ShortTermRentalLicense is issued. If a business is associated with the property, the Austin Code Program Specialist verifies the business information with the Secretary of State.
018 Secretary of State/verifyShortTermRentalInformation
The Austin Code Program Specialist verifies LLC, trust, or corporation information through the Secretary of State system.
019 Case Management System/createShortTermRentalLicense
The Austin Code Program Specialist creates the ShortTermRentalLicense in the Case Management System. The license includes Owner name, property address, name of short-term rental, number of bedrooms, contact information, issuance date, expiration date, OperatingLicenseNumber, and ShortTermRentalTypeDesignation (type 1, 2, 3, etc.).
020 Division Manager/activateShortTermRentalLicense
The Austin Code Program Specialist requests the Division Manager to activate the Short-Term Rental License (typically delivered in person). The Division Manager does the final review for the Short-Term Rental License before it's sent out.
021 Case Management System/activateShortTermRentalLicense
The Division Manager activates the ShortTermRentalLicense in the Case Management System (for both new and renewal licenses).
022 Austin Code Program Specialist/mailShortTermRentalLicenseToOwner
The Austin Code Program Specialist mails the Short-Term Rental License to the Short-Term Rental Applicant via regular mail.
023 Case Management System/updateShortTermRentalLicense
The Austin Code Program Specialist scans and attaches all ShortTermRentalApplication documents to the ShortTermRentalLicense in the Case Management System. The Austin Code Program Specialist notes any changes to Owner information, etc.
024 Austin Code Program Specialist/sendNewShortTermRentalsToCityController
The Austin Code Program Specialist sends a report monthly (typically at the beginning of the month) that lists the new ShortTermRentals for the previous month to the City Controller.
