Obtain City Legal Approval
Use Case Model

Scope: The Austin Code Officer creates an Affidavit, and then a SearchWarrant using the Affidavit's information. The Austin Code Officer submits the Affidavit and SearchWarrant to the Chain of Command for approval. The Chain of Command reviews the Affidavit and SearchWarrant, and either approves the documents or requests changes. If changes are required, the Austin Code Officer updates the Affidavit/SearchWarrant, and resubmits for approval. The submission process continues until the Chain of Command approves both the Affidavit and SearchWarrant. After approval from the Chain of Command, the Austin Code Officer submits the Affidavit and SearchWarrant to the City Legal Department for review. The City Legal Department reviews the documents and either approves or rejects the documents. If rejected, the use case is restarted with the Austin Code Officer creating the Affidavit and SearchWarrant.

Sequence Diagram

Req#Type/Interface - Requirement DescriptionPerformance
001 Austin Code Officer/createAffidavit
The Austin Code Officer creates an Affidavit, either search warrant, or event/probable cause, using CodeCase information.
002 Austin Code Officer/createSearchWarrant
The Austin Code Officer creates a SearchWarrant. SearchWarrant contains the summary of the Affidavit along with the same information on the Affidavit.
003 Chain of Command/reviewAffidavitAndSearchWarrant
The Austin Code Officer requests the Chain of Command reviews the Affidavit and Search Warrant. The Austin Code Officer delivers a print out of the Affidavit and Search Warrant. The Chain of Command either approves the documents or requests changes. If approved, the Chain of Command returns the Documents.
004 City Legal Department/reviewAffidavitAndSearchWarrant
The Austin Code Officer requests the City Legal Department review the Affidavit and SearchWarrant (either hand-delivered or through email). City Legal Department reviews document to ensure probable cause is met. If probable cause is met the documents are approved.
