Inspect Waste Hauler
Use Case Model

Scope: The Austin Code Officer performs daily inspections of Waste Haulers starting in February of each calendar year. The Austin Code Officer performs an Inspection of the Waste Hauler. The Austin Code Officer then requests the Austin Code Program Specialist to verify the WasteHaulerLicense. The Austin Code Program Specialist uses the system to verify the WasteHaulerLicense (that information is accurate, license isn't expired, etc.). If the Austin Code Officer identifies a Violation, the Austin Code Officer determines whether a Warning is required or a Citation instead. If a Warning, the Austin Code Officer issues the Warning. If a Citation is merited, the Austin Code Officer begins the Issue Violation Citation use case. Seven business days after the initial inspection, the Austin Code Officer performs a follow-up inspection to determine if the Violation has been resolved.

Req#Type/Interface - Requirement DescriptionPerformance
001 Austin Code Officer/performInspection
If the violation concerns hazardous conditions, the Austin Code Officer applies Personal Protective Equipment (PPE). The Austin Code Officer inspects the Property. The Austin Code Officer inspects from the exterior (initially from the Right of Way) to the interior (if the interior is accessible and permission has been granted by legally interested party or someone in Care/Control of Property). The Austin Code Officer take Photographs and Measurements (hot water temperature, air quality, interior temperature, light meter, ground-fault circuit interrupter, gas meter, carbon monoxide, smoke detector, fences, grass height, driveway and walkways, check impervious cover, etc.) as needed. As needed, the Austin Code Officer performs research to identify the Property Owner, Offense, ViolationType, OwnerInformation, and relevant PropertyInformation (GIS system, ARR Site, Annexation Information, Amanda Permit History, Site Plan, TCAD/WCAD for OwnerHistory and DeedHistory, CC&B, ZoningHistory, CodeOfOrdinances, etc.). The Austin Code Officer interviews relevant parties. For Waste Hauler Inspection: The Austin Code Officer makes contact with driver, visually inspects vehicle for decals on door, asks driver about awareness requirement for waste hauler license, ensures driver is operating within City of Austin (outside of city exempt), double check the Waste Hauler license, check for safety equipment violations, driver's license of driver, and take photos of truck (including both doors and front license plate). If a short-term rental complaint, the Austin Code Officer responds at a faster time than other complaints. On Fridays and Saturdays, the Austin Code Officer performs inspection during day, ands from 6:00 P.M. - 2:00 A.M. On Sundays, inspections occur from 8:00 A.M. - 12:00 P.M.
002 Austin Code Program Specialist/verifyWasteHaulerLicense
The Austin Code Officer requests the Austin Code Program Specialist to verify the WasteHaulerLicense (that the Waste Hauler has a license and it isn't expired). The Austin Code Program Specialist verifies the WasteHaulerLicense in the system.
003 Case Management System/queryWasteHaulerLicense
The Austin Code Program Specialist uses the system to verify the WasteHaulerLicense.
004 Austin Code Officer/issueWarning
The Austin Code Officer issues a Warning to the Violator. Note: Warnings are non-legal notices. A NoticeOfViolation is still required after a Warning is issued.
005 Austin Code Officer/performFollowUpInspection
Follow-up Inspection for Lodging License: The Austin Code Officer checks for GFCI outlets, smoke detectors (set off and sound), trip hazards, cracked windows, broken windows, weather stripping, temperature of hot water, pest infestation, secured balcony railing, overhead light inspection outside, loose shower spouts, secured toilet, faucet handles, locks on windows (if three feet up). Ten percent of the total rooms are inspected. Checklist: Structure Maintenance Report. Follow-up inspection for Waste Hauler: A WasteHaulerApplication is turned in, and the Austin Code Officer works with the Austin Code Program Specialist to ensure the WasteHaulerApplication is complete. The Austin Code Officer performs an Inspection to determine if the Waste Hauler Violation was resolved. If Illegal Dumping, the Austin Code Officer ensures property is abated. The Austin Code Officer takes photographs to document the Inspection.
