Close Code Compliant Case
Use Case Model

Scope: The Austin Code Officer updates the CodeComplaintCase with the results of their Inspection and the Case Management System then closes the CodeComplaintCase. If Complainant information is available and contact warranted, the Austin code Officer contacts the Complainant with their findings.

Close Code Compliant Case
Sequence Diagram

Req#Type/Interface - Requirement DescriptionPerformance
001 Case Management System/updateCodeComplaintCase
The user updates the CodeComplaintCase with information on the Complaint. If additional information provided by Complainant it's added to CodeComplaintCase. The Austin Code Officer adds Inspection results to CodeComplaintCase including Measurements and Photographs. If a DoorHanger was placed on the Property the Austin Code Officer adds notes for DoorHanger. If other government agencies are contacted the Austin Code Officer adds notes about agency contacted and services requested.
002 Case Management System/closeCodeComplaintCase
The Case Management System updates the CodeComplaintCase's status to closed.
003 Austin Code Officer/contactComplainantWithFindings
The Austin Code Officer contacts the Complainant, method of contact varies based on contact information available, to notify of whether a Violation was found, or not. The Austin Code Officers provides an overview of the Code Process and Expected Actions, along with Expected Resolution Time.
