Provide Community Outreach Assistance
Use Case Model

Scope: The Austin Code Officer identifies whether the responsible party has a HardshipReason. If the responsible party is unable to correct the Violation due to hardship, the Austin Code Officer identifies three assistance resources from the CodeResourceList. The Austin Code Officer updates the CodeViolationCase with notes of the resources identified. If the responsible party does not qualify for assistance with the provided assistance resources, the Austin Code Officer prepares the CommunityOutreachForm. The Austin Code Officer requests the Chain of Command approve the CommunityOutreachForm. If the CommunityOutreachForm is not approved, the Austin Code Officer continues to prepare/modify the CommunityOutreachForm until it is approved by the Chain of Command. Once the CommunityOutreachForm is approved, the Austin Code Officer requests the Public Information Office Admin provide assistance with the case. The Public Information Office Admin reviews the CommunityOutreachForm and determines if assistance services are approved. If assistance services are approved, the Public Information Office Admin begins the Provide Assistance Services process. The Public Information Office Admin communicates with the Austin Code Officer concerning the CommunityOutreachForm and any ServiceCases created for the responsible party. The Austin Code Officer updates the CodeViolationCase with notes on the assistance provided.

Provide Community Outreach Assistance
Sequence Diagram

Req#Type/Interface - Requirement DescriptionPerformance
001 Austin Code Officer/identifyHardshipQualificationForAssistance
The Austin Code Officer identifies hardship qualification. Hardship criteria include: low income (based on NationalMedianIncome), age (elderly or children), and SpecialNeeds (disabled, mental needs, visual problems, wheelchair-bound, etc.).
002 Austin Code Officer/identifyAssistantCorrectiveResources
The Austin Code Officer identifies three CorrectiveResources for the Owner/Tenant using the CodeResourceList. The Austin Code Officer provides the CorrectiveResources to the Owner/Tenant.
003 Case Management System/updateCodeViolationCase
The user updates the CodeViolationCase with relevant information to the case. If the CodeViolationCase generated a StopWorkOrder, the StopWorkOrder is attached to the CodeViolationCase. The user attaches a copy of the DiagramOfWorkRequired. If a board and secure occurs, a BoardAndSecureBill is attached to the CodeViolationCase by Austin Code Admin. If a sign was removed, the Austin Code Officer adds the sign and information to the CodeViolationCase. Additionally, if CautionBarriers were placed, information on the CautionBarriers is added to the CodeViolationCase. If property information was updated, the Case Management System includes updated property information. The user updates the CodeViolationCase with information relevant to the change. This includes, but is not limited to, BoardAndSecureBill, PropertyAbatementBill, FollowUpNotes, photographs (boarded, non-boarded, posted notifications, placards), emails, PhoneCallSummary, other relevant information and inter-department communications. If a ServiceCase is created, the Public Information Office Admin adds notes to the CodeViolationCase including the CitizenNeed and ServicesRecommended for ServiceCase. If CorrectiveResources are identified, the user updates the CodeViolationCase with a list of CorrectiveResources provided (three resources) and notes of contact with Owner/Tenant. HardshipReason is added to the CodeViolationCase. If new violations are identified, the user adds the new violation to the Case Management System and the violation is attached to the original CodeComplaintCase. If a TitleSearch is completed, the TitleSearch is added to the CodeViolationCase. If additional LegallyInterestedParties are identified, the parties are added to the CodeViolationCase. If a NoticeOfViolation is generated, the NoticeOfViolation is added to the CodeViolationCase. If posted at the property, photographs are attached to the CodeViolationCase. If the NoticeOfViolation is delivered and a certified mail receipt is received, the receipt is added to the CodeViolationCase. If a citation is created, the Case Management System attaches the citation to the CodeViolationCase. Additionally, the user scans and uploads a copy of the physical citation to the Case Management System. If an AdministrativeHearingWarning is issued, the user updates the CodeViolationCase with information on the issued AdministrativeHearingWarning. If an AdministrativeHearingCitation is issued, the user updates the CodeViolationCase with the AdministrativeHearingCitation, and a LegalCase is created. The user scans and uploads returned liens to the CodeViolationCase. If a lien is filed for the property, the TRVNumber is added to the CodeViolationCase. The Austin Code Officer updates the CodeViolationCase with information concerning Violation extensions, written request, reason for approval/denial, etc.
004 Austin Code Officer/prepareCommunityOutreachForm
The Austin Code Officer completes the CommunityOutreachForm using Owner/Tenant information and HardshipReason. If changes are required to the CommunityOutreachForm, the Austin Code Officer modifies the form.
005 Chain of Command/approveCommunityOutreachForm
The Austin Code Officer requests the Chain of Command review the CommunityOutreachForm and approve. Chain of Command primarily checks the CommunityOutreachForm for completeness.
006 Public Information Office Admin/requestAssistanceWithCase
The Austin Code Officer requests the Public Information Office Admin to request assistance with the case. The Austin Code Officer provides the CommunityOutreachForm. The Public Information Office Admin reviews the CommunityOutreachForm (including CodeViolationCase History). The Public Information Office Admin determines if Citizen qualifies for CommunityOutreach. The Public Information Office Admin staffs the Case with the Chain of Command. If the Case is denied, the Case is closed, Case Management System updated, and Austin Code Officer notified. If approved, the Provide Assistance Services process begins.
007 Public Information Office Admin/evaluateAndCommunicateWithAustinCodeOfficer
The Public Information Office Admin notifies the Austin Code Officer of the Services provided to the Citizen.
