Issue Waste Hauler License
Use Case Model

Scope: Applications are always sent in November of calendar year (certified mail) to all Waste Haulers to renew. Containers are renewed every 3-months, Tonnage Reports sent every 6-months, Vehicles once a year unless adding new vehicle, then ad hoc. If a WasteHaulerApplication is not received the Waste Hauler contacts Austin 311 for a WasteHaulerApplication. Austin 311 requests the Austin Code Program Specialist provide the WasteHaulerApplication to the Waste Hauler. Alternatively, the Waster Hauler downloads the WasteHaulerApplication. The Waste Hauler completes the WasteHaulerApplication. The Waste Hauler submits the WasteHaulerApplication either by mail to Austin Code Finance, or in person to an Austin Code Program Specialist. If Austin Code Finance receives the WasteHaulerApplication they start the Process Payment use case and forward the WasteHaulerApplication to an Austin Code Program Specialist. If an Austin Code Program Specialist receives the application they review the WasteHaulerApplication, provide a Receipt to the Waste Hauler, and then forwards the Payment (Check) to Austin Code Finance to start the Process Payment use case. The Austin Code Program Specialist reviews the WasteHaulerApplication. If the Check amount is incorrect the Austin Code Program Specialist requests the Waste Hauler submit a new Check. If the WasteHaulerApplication requires changes the Austin Code Program Specialist requests the Waste Hauler updates the application. The WasteHaulerApplication is reviewed until the application is paid fully and complete. The Austin Code Program Specialist uses the Case Management System to create a WasteHaulerLicense which includes VehicleLicenses. The Austin Code Program Specialist requests Austin Resource Recovery verifies the TonnageReport included in the WasteHaulerApplication. Austin Resource Recovery contacts the Waste Hauler for verification. When the TonnageReport is verified Austin Resource Recovery returns the TonnageReport signed. When the TonnageReport is returned signed the Austin Code Program Specialist issues the WasteHaulerLicense to the Waste Hauler (print licenses and mail via certified mail). Printed licenses include WasteHaulerID, VehicleID, LicensePlate, and ExpirationDate. Notification to Waste Hauler includes a Letter explaining Waste Hauler Process. When the CertifiedMailNumber (receipt of delivery) is received the Austin Code Program Specialist adds the Receipt to the Letter and uploads to the Case Management System.

Issue Waste Hauler License
Sequence Diagram

Req#Type/Interface - Requirement DescriptionPerformance
001 Austin 311/provideWasteHaulerApplication
The Waste Hauler contacts 311 and requests a waste hauler application be sent to them. They provide their name, phone number, email and physical address.
002 Austin Code Program Specialist/provideWasteHaulerApplication
311 Contacts the Austin Code Program Specialist, and indicates a potential Waste Hauler has requested an application. The Austin Code Program Specialist will call the Waste Hauler, confirming the information provide, and then sends via US Postal a Waste Hauler Application. Note: the Waste Hauler can request the application be emailed, but the Austin Code Specialist will still send this via US Postal as well. This is sent certified mail.
003 Waste Hauler/downloadWasteHaulerApplication
The Waste Hauler uses the City Website, using the Austin Code Department to locate a Waste Hauler application. The Waste Hauler fills out the information as required.
004 Waste Hauler/completeWasteHaulerApplication
The Waste Hauler fills out the Waste Hauler application and then locates a notary to validate and annotate the signature.
005 Austin Code Finance/processWasteHaulerApplication
The Waste Hauler sends the application to the Austin Code Finance by mail. Austin Code Finance makes copies of the front page , the check and remittance forms and then starts the Process Payment use case.
006 Austin Code Program Specialist/processWasteHaulerApplication<
The Waste Hauler delivers the application in person to a "finance approved" staff member (Austin Code Program Specialist). The Austin Code Program Specialist reviews the form (assuring notarization, signatures from responsible parties), and if everything is completed, they make copies of the front page, the check and remittance forms. The Austin Code Program Specialist records the payment and provides a receipt to the Waste Hauler.
007 Austin Code Program Specialist/reviewWasteHaulerApplication
The Austin Code Program Specialist reviews the WasteHaulerApplication using a checklist (is notarized, all information is complete, insurance, inspection, check, container information, signature for Austin Code Program Specialist, how many licenses issued: 1 per vehicle). verifying presence of insurance forms, inspection reports, tonnage reports, and vehicle information.
008 Waste Hauler/submitNewCheck
The Austin Code Program Specialist determines the amount paid was less than required and request additional funds. Note the Waste Hauler does not need to resubmit the application in this case.
009 Waste Hauler/updateWasteHaulerApplication
The Austin Code Program Specialist determines the application was incorrect. And requests the Waste Hauler modify, and resubmit the application. If the Waste Hauler has not provided the required reports, they can simply email those to the Austin Code Program Specialist.
010 Case Management System/createUpdateWasteHaulerLicense
The AustinCode Program Specialist uses the Case Management system to create a waste hauler vehicle license. They provide the following information: Name of company, contact name, contact information, vehicle information (license place, VIN, Gross vehicle weight, make, model, year). The license produced is color coded - (blue: recycle/compost, Orange:Landfill ...combos of both are always coded as Landfill. ) NOTE: the Landfill color codes change annually. The AustinCode Program Specialist uses the Case Management system to update the waste hauler vehicle license. They provide the following information: Name of company, contact name, contact information, vehicle information (license place, VIN, Gross vehicle weight, make, model, year). This normally occurs when a new vehicle for an existing company is added. The Austin Code Program Specialist updates the WasteHaulerLicense with the CertifiedMailReceipt for the distributed WasteHaulerLicenses.
011 Austin Resource Recovery/verifyTonnageReport
The Austin Code Program Specialist requests Austin Resource Recovery verify the TonnageReport included in the WasteHaulerApplication. Austin Resource Recovery contacts the Waste Hauler, verifies the TonnageReport, and returns a signed TonnageReport.
012 Waste Hauler/verifyTonnageReport
Austin Resource Recovery contacts the Waste Hauler and requests verification of the TonnageReport.
013 Austin Code Program Specialist/distributeWasteHaulerLicenses
The Austin Code Specialist send via US Postal Certified mail, the Waste Hauler licenses.
014 Case Management System/createUpdateWasteHaulerLicense
The AustinCode Program Specialist uses the Case Management system to create a waste hauler vehicle license. They provide the following information: Name of company, contact name, contact information, vehicle information (license place, VIN, Gross vehicle weight, make, model, year). The license produced is color coded - (blue: recycle/compost, Orange:Landfill ...combos of both are always coded as Landfill. ) NOTE: the Landfill color codes change annually. The AustinCode Program Specialist uses the Case Management system to update the waste hauler vehicle license. They provide the following information: Name of company, contact name, contact information, vehicle information (license place, VIN, Gross vehicle weight, make, model, year). This normally occurs when a new vehicle for an existing company is added. The Austin Code Program Specialist updates the WasteHaulerLicense with the CertifiedMailReceipt for the distributed WasteHaulerLicenses.
