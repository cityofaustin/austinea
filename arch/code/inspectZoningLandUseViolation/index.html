Inspect Zoning Land Use Violation
Use Case Model

Scope: The Austin Code Officer determines the specific Violation. The Austin Code Officer uses the Case Management system to identify PropertyInformation. The Austin Code Officer completes additional research on LandHistory. If the Violation is a contractor performing work without a permit the Inspector issues a StopWorkOrder. StopWorkOrders are issued at the Austin Code Officer's discretion based on the progress of the work, impact to structure, and public safety. The Austin Code Officer creates a StopWorkOrder, then uses the Case Management System and creates a StopWorkOrderCoverLetter. The Austin Code Officer requests the Austin Code Program Specialist issue a StopWorkOrder, and send it through certified and regular mail. The Austin Code Program Specialist updates the CodeViolationCase with notes on the StopWorkOrder. If the Violation requires a suspension or revocation of a permit/certificate/etc. the Austin Code Officer requests Planning or Development Review research potential suspension or revocation of the appropriate permit/certificate/etc. The Austin Code Officer follows up with Planning or Development Review on the revocation until a decision or resolution is determined. If the Violation is a SignViolation the Austin Code Officer removes the sign. The Austin Code Officer updates the CodeViolationCase with the results of the investigation.

Inspect Zoning Land Use Violation
Sequence Diagram

Req#Type/Interface - Requirement DescriptionPerformance
001 Austin Code Officer/determineSpecificViolation
The Austin Code Officer determines the specific Zoning or Land Use Violation. Possible violation types include work without permit (mechanical, electrical, plumbing, and building). Other land use violations include (site plans, incorrect/changed/missing certificate of occupancy, site plan exemptions, change of use, compatibility standards, permits and licensing, etc.).
002 Case Management System/identifyPropertyInformation
The Austin Code Officer requests the Case Management System to look up certificate of occupancy, site plan, building permits, electrical permits, plumbing permits, mechanical permits, other permits, change of use, plan review, variances, neighborhood plans, conditional overlays, licensing, and CaseHistory.
003 Austin Code Officer/performAdditionalLandHistoryResearch
The Austin Code Officer performs research using the following systems from the CodeResourceList: TCAD, WCAD, ARR GIS Maps, CC&B, GIS Development Map (maps and neighborhood plans), Secretary of State (SOS), Travis County Public Records, Lexis Nexus, Department of Motor Vehicles, Department of Public Safety (DPS), title search, Case Management System, Austin History Center, Google Earth, physical file review (at the OTC), and historic aerial photographs. If a ShortTermRental Violation the Austin Code Officer performs research on Short Term Rental Websites (vacation rental research, Air BNB, BBRO, etc.). If the Property is located on a Short Term Rental Website a NoticeOfViolation is sent without requiring on-site inspection.
004 Austin Code Officer/createStopWorkOrder
The Inspector creates a StopWorkOrder using the CodeViolationCase information and posts on the Property (on the structure or premise). The Inspector also provides a copy of the StopWorkOrder to the Owner/Tenant in person, if available.
005 Case Management System/createStopWorkOrderCoverLetter
The Austin Code Officer uses the Case Management System to generate a StopWorkOrderCoverLetter using the CodeViolationCase information.
006 Austin Code Program Specialist/issueStopWorkOrder
The Austin Code Officer requests the Austin Code Program Specialist send a certified and regular mail copy of the StopWorkOrder to the Owner or Responsible Party.
007 Case Management System/updateCodeViolationCase
The user updates the CodeViolationCase with relevant information to the case. If the CodeViolationCase generated a StopWorkOrder, the StopWorkOrder is attached to the CodeViolationCase. The user attaches a copy of the DiagramOfWorkRequired. If a board and secure occurs, a BoardAndSecureBill is attached to the CodeViolationCase by Austin Code Admin. If a sign was removed, the Austin Code Officer adds the sign and information to the CodeViolationCase. Additionally, if CautionBarriers were placed, information on the CautionBarriers is added to the CodeViolationCase. If property information was updated, the Case Management System includes updated property information. The user updates the CodeViolationCase with information relevant to the change. This includes, but is not limited to, BoardAndSecureBill, PropertyAbatementBill, FollowUpNotes, photographs (boarded, non-boarded, posted notifications, placards), emails, PhoneCallSummary, other relevant information and inter-department communications. If a ServiceCase is created, the Public Information Office Admin adds notes to the CodeViolationCase including the CitizenNeed and ServicesRecommended for ServiceCase. If CorrectiveResources are identified, the user updates the CodeViolationCase with a list of CorrectiveResources provided (three resources) and notes of contact with Owner/Tenant. HardshipReason is added to the CodeViolationCase. If new violations are identified, the user adds the new violation to the Case Management System and the violation is attached to the original CodeComplaintCase. If a TitleSearch is completed, the TitleSearch is added to the CodeViolationCase. If additional LegallyInterestedParties are identified, the parties are added to the CodeViolationCase. If a NoticeOfViolation is generated, the NoticeOfViolation is added to the CodeViolationCase. If posted at the property, photographs are attached to the CodeViolationCase. If the NoticeOfViolation is delivered and a certified mail receipt is received, the receipt is added to the CodeViolationCase. If a citation is created, the Case Management System attaches the citation to the CodeViolationCase. Additionally, the user scans and uploads a copy of the physical citation to the Case Management System. If an AdministrativeHearingWarning is issued, the user updates the CodeViolationCase with information on the issued AdministrativeHearingWarning. If an AdministrativeHearingCitation is issued, the user updates the CodeViolationCase with the AdministrativeHearingCitation, and a LegalCase is created. The user scans and uploads returned liens to the CodeViolationCase. If a lien is filed for the property, the TRVNumber is added to the CodeViolationCase. The Austin Code Officer updates the CodeViolationCase with information concerning Violation extensions, written request, reason for approval/denial, etc.
008 Planning Development And Review/researchPotentialSuspensionOrRevocation
The Austin Code Officer either Emails or calls the Planning and Development Review specific work group (Electrical, Mechanical, Plumbing, Building, etc.) to review Investigator findings and determine if a revocation is appropriate. The Austin Code Officer provides the Property and details from the CodeViolationCase.
009 Austin Code Officer/followupWithPlanningAndDevelopmentReviewOnRevocation
The Austin Code Officer follows up (phone or email) with Permitting department to confirm revocation of appropriate permit/certificate. Austin Code Officer performs follow up based on Severity and Disposition of Issue.
010 Austin Code Officer/removeSign
The Austin Code Officer removes the Sign from the Right of Way. The Austin Code Officer discards the Sign. The Austin Code Officer takes note of the Sign properties (Location, SignDescription).
