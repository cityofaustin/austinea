Issue Signs and Billboard License
Use Case Model

Scope: If relocating a billboard, the Billboard Owner contacts Planning or Development and Review (PDR), and requests they provide the SignRegistrationApplication. The Billboard Owner completes the SignRegistrationApplication and then sends it to the Planning or Development and Review group for processing. After the Planning or Development and Review group processes the SignRegistrationApplication, a SignRegistration is created. The Planning or Development and Review group contacts the Austin Code Program Specialist to approve the SignRegistration. The Austin Code Program Specialist accesses the Case Management System and approves the SignRegistration. After approval of the SignRegistration, Planning or Development and Review completes the SignRegistration in the Case Management System. A BillboardInvoice is created by the Austin Code Program Specialist for the Billboard Owner. Alternatively, the Case Management System can create the BillboardInvoice. The Billboard Owner contacts Austin Code Finance to record payment for the invoice. The SignRegistration is updated in the Case Management System and the Case Management System is accessed to print a Billboard Receipt and the SignRegistration. Updates are either completed by the Austin Code Program Specialist per billboard, or in batches by the Case Management System. The Austin Code Program Specialist sends the SignRegistration to the Billboard Owner through regular mail. If the billboard was moved, the Austin Code Program Specialist requests an Austin Code Officer to perform a follow-up inspection and take photographs of the moved billboard.

Sequence Diagram

Req#Type/Interface - Requirement DescriptionPerformance
001 Planning or Development and Review/provideSignRegistrationApplication
The Billboard Owner contacts PDR and requests a SignRegistrationApplication for a billboard move. PDR returns the SignRegistrationApplication to the Billboard Owner. Note: Follow up with PDR for more information about SignRegistrationApplication.
002 Billboard Owner/completeSignRegistrationApplication
The Billboard Owner completes the SignRegistrationApplication.
003 Planning or Development and Review/processSignRegistrationApplication
The Billboard Owner requests PDR to process the SignRegistrationApplication, and PDR reviews it. Note: Follow up with PDR on specifics.
004 Austin Code Program Specialist/approveSignRegistration
PDR contacts the Austin Code Program Specialist to approve the SignRegistration and billboard move (email: PermitNumber is required).
005 Case Management System/approveSignRegistration
The Austin Code Program Specialist uses the Case Management System to approve the SignRegistration based on the PermitNumber. Once approved, the Case Management System moves the SignRegistration to the PDR group for additional work.
006 Planning or Development and Review/completeSignRegistration
PDR does additional work to complete the SignRegistration. Note: Review with PDR for more information.
007 Austin Code Program Specialist/createBillboardInvoiceAndSendToOwner
The Austin Code Program Specialist creates a BillboardInvoice and sends to the Billboard Owner through regular mail. BillboardInvoice include Billboard Owner's name, Agent's name, permit start date, permit end date, permit number, address of billboard and billboard dimensions. Notification is typically sent in October of the year the permits expire (December 31).
008 Case Management System/createBillboardInvoice
***Needs Documentation***
009 Austin Code Finance/recordPayment
Austin Code Finance receives a BillPayment and deposits the BillPayment within 24 hours of receipt. Austin Code Finance records the BillPayment in AIMs, and creates a physical copy of the BillPayment and returns a receipt.
010 Case Management System/updateSignRegistration
The Case Management System updates the SignRegistration by applying payment value to Sign/Billboard and the expiration date is updated to end of next calendar year (December 31).
011 Case Management System/printBillboardReceipt
The Case Management System prints the BillboardReceipt, which includes payment date, receipt number, invoice number, company name, billed-to address, phone number, payment method, payment received, amount applied, cash returned, comments with check number, department name, who issued the receipt, FAOCode, fee description, internal reference number, address, permit/case number, amount and total.
012 Case Management System/printSignRegistration
The Case Management System prints the SignRegistration, which includes the Billboard Owner's name, Agent's name, permit start date, permit end date, permit number, address of billboard and billboard dimensions.
013 Austin Code Program Specialist/sendSignRegistrationToOwner
The Austin Code Program Specialist mails the Billboard/Sign Registration and receipts to the Billboard Owners through regular mail.
014 Austin Code Officer/performFollowUpInspection
Follow-up Inspection for Lodging License: The Austin Code Officer checks for GFCI outlets, smoke detectors (set off and sound), trip hazards, cracked windows, broken windows, weather stripping, temperature of hot water, pest infestation, secured balcony railing, overhead light inspection outside, loose shower spouts, secured toilet, faucet handles, locks on windows (if three feet up). Ten percent of the total rooms are inspected. Checklist: Structure Maintenance Report. Follow-up inspection for Waste Hauler: A WasteHaulerApplication is turned in, and the Austin Code Officer works with the Austin Code Program Specialist to ensure the WasteHaulerApplication is complete. If Illegal Dumping, the Austin Code Officer ensures property is abated and takes photographs.
