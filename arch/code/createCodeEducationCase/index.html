Create Code Education Case
Use Case Model

Scope: If an Austin Code Officer identifies a Violation that can be corrected easily the Austin Code Officer notifies the Responsible Party (through a stop and knock). The Austin Code Officer creates a CodeEducationCase in the Case Management System. 24-48 hours later the Austin Code Officer performs an Inspection to see if the Violation was corrected. If the Violation was not corrected, the Austin Code Officer initiates the Create Code Complaint use case. If the Violation was corrected, or after the CodeComplaintCase is created, the Austin Code Officer uses the Case Management System to close the CodeEducationCase.

Create Code Education Case
Sequence Diagram

Req#Type/Interface - Requirement DescriptionPerformance
001 Austin Code Officer/resolveViolationBeforeCodeViolationCaseCreated
The Austin Code Officer notifies the Responsible Party of the Violation. Notification is provided in person through a stop and knock.
002 Case Management System/createCodeEducationCase
The Austin Code Officer creates a CodeEducationCase in the Case Management System. CodeEducationCase identifies the Violation and Property.
003 Austin Code Officer/performInspection
If the violation concerns hazardous conditions, the Austin Code Officer applies Personal Protective Equipment (PPE). The Austin Code Officer inspects the Property. The Austin Code Officer inspects from the exterior (initially from the Right of Way) to the interior (if the interior is accessible and permission has been granted by legally interested party or someone in Care/Control of Property [follow up with City Legal for verbiage]). The Austin Code Officer take Photographs and Measurements (hot water temp, air quality, interior temp, light meter, ground fault circuit interrupter, gas meter, carbon monoxide, smoke detector, fences, grass height, driveway and walkways, check impervious cover) as needed. The Austin Code Officer performs research, as needed, to identify the Property Owner/Offense/ViolationType, OwnerInformation, and relevant PropertyInformation (GIS system, ARR Site, Annexation Information, Amanda Permit History, Site Plan, TCAD/WCAD for OwnerHistory and DeedHistory, CC&B, ZoningHistory, CodeOfOrdinances, etc.). The Austin Code Officer interviews relevant parties. For Waster Hauler Inspection: The Austin Code Officer makes contact with driver, visually inspects vehicle for decals on door, ask driver about awareness requirement for waste hauler license, ensure driver is operating in the city of Austin (outside of city exempt), double check that doesn't have Waste Hauler license (separate method to program specialist), check for safety equipment violations, driver's license of driver, take photos of truck (both doors, front license plate). If a Short Term Rental Complaint the Austin Code Officer responds at a faster time than other Complaints. On Fridays and Saturdays Austin Code Officer performs inspection during day, plus from 6:00PM - 2:00AM. Sundays inspections occur 8:00AM - 12:00PM.
004 Case Management System/closeCodeEducationCase
The Austin Code Officer updates the CodeEducationCase to close it.
