Board and Secure Premise
Interaction Overview

Scope: The Austin Code Officer creates a DiagramOfWorkRequired using the CodeViolationCase's information (Property). The Austin Code Officer requests the Contractor boards and secures the premise. The Contractor returns a BoardAndSecureBill to Austin Code. The Austin Code Officer updates the CodeViolationCase with the DiagramOfWorkRequired. After the Property has been boarded and secured, the Austin Code Officer inspects the property--typically within 24 hours from the board and secure. After the board and secure, if a BoardAndSecureBill hasn't been received before the mandated notification, the Austin Code Program Specialist requests the Contractor provide a bill. The Austin Code Program Specialist requests the Austin Code Officer verify the board and secure. The Austin Code Program Specialist notifies the Owner of the board and secure through certified and regular mail. The Austin Code Officer posts the NoticeOfViolation within 10 days of work being completed and takes photographs of the posting. After posting the NoticeOfViolation, Austin Code pays the Contractor and bills the Owner. If the Owner pays the board and secure, the use case ends.

If the Owner does not pay, no homestead exemption exists and a lien is not already placed on the property, Austin Code begins the Place Lien on Property use case.

If fencing is used to board and secure the property, a new billing cycle occurs every 3 months to pay the Contractor and bill the Owner.
