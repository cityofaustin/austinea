Verify Board and Secure
Use Case Model

Scope: After the board and secure, if a BoardAndSecureBill hasn't been received before the mandated notification, the Austin Code Program Specialist requests the Contractor provide a bill. The Austin Code Program Specialist requests the Austin Code Officer verify the board and secure. The Austin Code Program Specialist notifies the Owner of the board and secure through certified and regular mail. The Austin Code Officer posts the NoticeOfViolation within 10 days of work being completed and takes photographs of the posting.

Verify Board and Secure
Sequence Diagram

Req#Type/Interface - Requirement DescriptionPerformance
001 Contractor/provideBill
The Austin Code Admin contacts the Contractor and requests the BoardAndSecureBill per the CodeViolationCase's property information.
002 Austin Code Officer/verifyBoardAndSecureCompletion
The Austin Code Admin requests the Austin Code Officer verify that the BoardAndSecureBill matches the results of their previous inspection.
003 Austin Code Program Specialist/notifyOwnerOfBoardAndSecure
The Austin Code Admin is notified of the board or secure by the bill arriving from the Contractor for work completed or the NoticeOfViolation is created. The Austin Code Admin proofreads, prints and mails (regular and certified) notification to the Owner. International Property Maintenance and Local Government Code requires certified mail be sent in concordance with current State of Texas laws.
004 Austin Code Officer/postBoardAndSecureNotice
The Austin Code Officer posts the NoticeOfViolation at the Property. Alternatively, the NoticeOfViolation may be delivered by hand. Regardless of method posted/delivered the Austin Code Officer takes photographs of the NoticeOfViolation at the Property.
