Develop Building Standards Commission Case
Use Case Model

Scope:

A CodeViolationCase that requires review by the Building and Standards Commission initiates the Verify Title use case. An Austin Code Officer requests a TitleSearch. Once the TitleSearch is returned the Austin Code Officer updates the CodeViolationCase.

After the TitleSearch the Create BSC Packet use case begins. The Austin Code Officer requests BSCMaps for the CodeViolationCase, and then assembles a BSCPacket. The BSCPacket receives multiple levels of review including the Austin Code Officer's direct Chain of Command, the Assistant Division Manager, and the Austin Code Legal Officer. If changes are required at any point the BSCPacket is returned to the Austin Code Officer for updates. Once approved, the Austin Code Legal Coordinator coordinates a BSCPacket case review.

The Perform Building Standards Commission Review use case starts after the BSCPacket is approved. The Austin Code Legal Coordinator creates a BSCAgenda, develops a BSCPresentation, and conducts pre-meetings in preparation for the Building and Standards Commission. The Building and Standards Commission reviews the BSCPacket and makes a BSCDetermination: defer a decision or accept the recommendation. If a decision is deferred, the Perform Building Standards Commission Review use case restarts at the deferred time.

  • If the decision is accepted the Building and Standards Commission may opt to alter the recommendation, or keep the original recommendation. An Order is created, signed, and notarized. Orders are for repairs, or demolition, and specify a time frame to be completed. If the Responsible Party doesn't comply to the Order the LegalCase is brought back to the Building and Standards Commission for review.

  • If the Responsible Party doesn't display an initiative to comply to the Order, and the Order is for a demolition the Austin Code Legal Coordinator coordinates the demolition process with Public Works, notifies utilities to disconnect services, and requests verification of the demolition by an Austin Code Officer.

If the Property is within compliance and all penalties are paid for the Order, the LegalCase is closed. If not, the Perform Building Standards Commission Review use case restarts.
