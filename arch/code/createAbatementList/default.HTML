Create Abatement List
Use Case Model

Scope: When a Property requires abatement, the Austin Code Officer requests the abatement. If a Contractor completes the abatement, the Austin Code Officer creates a WorkRequiredList in the Case Management System, and then assembles a WorkRequiredListPacket. The Austin Code Officer submits the WorkRequiredListPacket to the Chain of Command for approval in person. Alternatively, the Chain of Command approves the WorkRequiredList in the Case Management System. The approval process loops until the WorkRequiredList is approved by the Chain of Command. The Austin Code Program Specialist uses the Case Management System to identify abatement cases and creates a CutListPacket for the Contractor. The Contractor picks up the CutListPacket. If the Community Court completes the abatement, the Austin Code Officer creates a CommunityCourtAbatement list, and notifies Community Court of the requested abatement. Once Community Court has been notified, the abatement process is complete.

Sequence Diagram

Req#Type/Interface - Requirement DescriptionPerformance
001 Case Management System/createWorkRequiredList
The Austin Code Officer requests the Case Management System create a WorkRequiredList. The Austin Code Officer provides: type of work, classification (1-4), square footage for mowing, cubic yards for removal of trash debris (over 22,000 square feet is a lower rate, and rates are defined per contract with the Contractor with a $190 minimum charge per square footage), number of tires and size (large or small), special fee (minimum fee of $250 per contract with Contractor includes hand removal instead of machine, bee removal, fill pools with dirt, etc.), notes, and Contact Name and phone number (Austin Code Officer, Officer's Immediate Supervisor and/or Austin Code Admin Manager). The Case Management System calculates the TotalCost based on square footage, admin fee, special fee, number of tires, cubic yards of trash, etc.
002 Austin Code Officer/createAbatementWorkRequiredListPacket
The Austin Code Officer creates a WorkRequiredListPacket (WorkRequiredList, Case information, CoverPage, WorkRequiredChecklist). The CoverPage includes: address, TCAD information, map (highlighted by Austin Code Officer for areas to abate/work), photograph of property, photograph of posting, green card (return receipt from mail), if transient camp, if APD is required, CaseNumber, and inspection detail from Case Management System (estimate of work).
003 Chain of Command/evaluateWorkRequiredList
The Austin Code Officer submits the WorkRequiredListPacket to the Chain of Command. The Chain of Command evaluates the WorkRequiredListPacket and either approves or denies it. If approved, the Chain of Command completes the WorkRequiredChecklist (initials checklist). If denied, the Chain of Command returns the list to the Austin Code Officer with notes requesting changes. If the WorkRequiredList estimate changes and exceeds $1000, the Austin Code Officer requests the Chain of Command approves the new estimate on the WorkRequiredList.
004 Case Management System/submitWorkRequiredListToChainOfCommand
The Austin Code Officer submits the WorkRequiredList to the Chain of Command through the Case Management System. The Case Management System adds the WorkRequiredList to the Chain of Command's queue in the Case Management System.
005 Case Management System/evaluateWorkRequiredList
The Chain of Command uses the system to evaluate the WorkRequiredList. If approved, the system adds the approved WorkRequiredList to the CutListQueue. If denied, the Case Management System routes the WorkRequiredList back to the Austin Code Officer's queue with notes on changes requested.
006 Case Management System/identifyPropertyAbatementCases
The Austin Code Program Specialist uses the Case Management System to identify pending abatement cases (CodeViolationCases, WorkRequiredList).
007 Austin Code Program Specialist/createCutListPacket
The Austin Code Program Specialist prints the abatement cases and assembles them into a CutListPacket. The Austin Code Program Specialist contacts the Contractor notifying them that a new CutListPacket is available for pickup. The Contractor picks up the
008 Contractor/pickUpCutListPacket
The Contractor is notified to pick up the CutListPacket. Alternatively, the Contractor periodically picks up the CutListPacket from the Austin Code Program Specialist.
009 Austin Code Officer/createCommunityCourtAbatementList
The Austin Code Officer creates a CommunityCourtAbatement list that contains location, extra notes on location, work requested, date requested, Austin Code Officer's name and number, and the requesting Department (Austin Code Department).
010 Austin Code Officer/notifyCommunityCourtOfRequestedAbatement
The Austin Code Officer emails the CommunityCourtAbatement to the Community Court.
