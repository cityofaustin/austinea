Issue Lodging License
Use Case Model

Scope: A Lodging Business receives a LodgingLicenseApplication from an Austin Code Officer. Alternatively, the Lodging Business contacts Austin 311 and requests a LodgingLicenseApplication. Austin 311 asks the Austin Code Program Specialist to provide the LodgingLicenseApplication to the Lodging Business. The Lodging Business then completes the LodgingLicenseApplication. The Lodging Business submits the LodgingLicenseApplication to the Austin Code Program Specialist for processing. The Austin Code Program Specialist creates a LodgingLicense in the Case Management System, and then creates a LodgingLicenseInvoice and sends to the Lodging Business. The Lodging Business either submits a Check to Austin Code Finance by mail or the Lodging Business submits a Check to the Austin Code Program Specialist. If by mail, the Austin Code Finance begins the Process Payment use case; otherwise, the Austin Code Program Specialist records the payment and begins the Process Payment use case. After payment is received, the Austin Code Program Specialist uses the Case Management System to print the LodgingLicense. The Austin Code Program Specialist mails the LodgingLicense to the Lodging Business, and the Inspect Lodging Business use case is initiated.

Issue Lodging License
Sequence Diagram

Req#Type/Interface - Requirement DescriptionPerformance
001 Lodging Business/completeLodgingLicenseApplication
If in person, the Austin Code Officer provides a LodgingLicenseApplication and requests the Lodging Business to complete and submit it; otherwise, the Lodging Business completes the application that the Austin Code program Specialist emailed them.
002 Austin 311/provideLodgingLicenseApplication
The Lodging Business contacts Austin 311 and requests a LodgingLicenseApplication be sent to them. They provide their name, phone number, email and physical address.
003 Austin Code Program Specialist/provideLodgingLicenseApplication
The Austin Code Program Specialist emails the LodgingLicenseApplication to the Lodging Business.
004 Austin Code Program Specialist/processLodgingLicenseApplication
The Lodging Business completes the application and emails it to the Austin Code Program Specialist. They provide the following information: establishment address, phone number, owner info, manager info, billing info, type (hotel, motel, rooming house or boarding house) and number of rooms.
005 Case Management System/createLodgingLicense
The Austin Code Program Specialist uses the Case Management System to create the LodgingLicense. They provide the following information: establishment address, phone number, owner info, manager info, billing info, type (hotel, motel, rooming house or boarding house) and number of rooms.
006 Austin Code Program Specialist/createLodgingLicenseInvoiceAndSendToLodgingBusiness
The Austin Code Program Specialist uses the Case Management System to generate an invoice. The invoice is sent to the Lodging Business via email or US Postal System (depending how the original application was submitted). The invoice includes the following information: establishment address, phone number, owner info, manager info, billing info, number of rooms, case number, current date, type (hotel, motel, rooming house or boarding house), expiration date, and the license cost.
007 Austin Code Program Specialist/recordPaymentReceived
The Austin Code Program Specialist records the BillPayment (invoice) within the Case Management System. The Austin Code Program Specialist returns a receipt.
008 Case Management System/printLodgingLicense
The Austin Code Program Specialist uses the Case Management System to print the LodgingLicense.
009 Austin Code Program Specialist/sendLodgingLicensetoLodgingBusiness
The Austin Code Program Specialist uses the US Postal Service to send the printed LodgingLicense to the billing address provided with the application information.
