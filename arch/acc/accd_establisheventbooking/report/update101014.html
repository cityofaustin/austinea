Perform Waste Management
Use Case

Use Case Scope

The Waste Collector collects waste from receptacles around the facility and then asks the Waste Sorter to sort and weigh the Waste. The Waste Sorter sorts and weighs the Waste and uses the system to input Waste Log Information. Upon saving this information, the system automatically sends an email notification to relevant staff, notifying them of the waste that was just sorted, weighed and logged. On a daily basis, the system evaluates Waste weight and quantity totals and if these exceed waste thresholds, an email notification is sent to the Waste Sorter to inform them that a Waste Vendor Pickup is required. The Waste Sorter calls the appropriate Waste Vendor to request a pickup and receives a projected Waste Pickup Date. The Waste Sorter uses the system to input the Waste Pickup Date, in the process creating a Waste Pickup Request. The Waste Vendor arrives at the facility and collects the waste, providing the Waste Sorter with a Waste Vendor Receipt. The Waste Sorter uses the system to search for the Waste Pickup Request and logs the Waste Pickup Date Actual value as well as scans and embeds a copy of the Waste Vendor Receipt. NOTE: Waste collection reports are generated on a monthly basis to compare Waste Weights and Waste Amounts with projected totals to ensure the facility maintains its LEED certification status.

Perform Waste Management Interface Descriptions

Type/InterfaceDescription
Waste Sorter/
sortWasteByType		 The Waste Collector collects waste from receptacles around the facility and asks the Waste Sorter to sort the Waste by Type (Recycle, Reuse, Incinerate, Landfill.)
Provide Safe And Clean Space/
logWasteInformation		 The Waste Sorter sorts and weighs all Waste items by Waste Type and then uses the system to enter Waste Log Information (Waste Item, Waste Log Date, Location, Waste Type [Recycle, Reuse, Incinerate, Landfill], Waste Weight, Waste Amount, Waste Disposal Method [Vendor Pickup/Delivery], Waste Vendor [Goodwill, State Surplus Store, Habitat for Humanity, TD Pallets, Austin Resource Recovery, FSG, Veolia, Clean Harbors])
Provide Safe And Clean Space/
sendWasteCollectionNotification		 The system sends an email Notification to the Sustainability Coordinator, Facility Manager, Maintenance Supervisor, Crew Supervisor, informing them of the Waste that has been collected, sorted and logged. The email includes all Waste Log Information (Waste Item, Waste Log Date, Location, Waste Type [Recycle, Reuse, Incinerate, Landfill], Waste Weight, Waste Amount, Waste Disposal Method [Vendor Pickup/Delivery], Waste Vendor [Goodwill, State Surplus Store, Habitat for Humanity, TD Pallets, Austin Resource Recovery, FSG, Veolia, Clean Harbors])
Provide Safe And Clean Space/
sendWastePickupNotification		 On a daily basis, the system examines Waste Log Information and sends notification to the Waste Sorter if the total Waste Weight or Waste Quantity per type exceeds the Waste Pickup Threshold, informing the Waste Sorter that a Waste Pickup Date is required. This process loops each day, until a Waste Pickup Date Actual value has been posted.
Waste Vendor/
pickupWaste		 The Waste Sorter contacts the appropriate Waste Vendor (using Waste Vendor Information) to request waste pickup. This request triggers the creation of a Waste Pickup Request Date. The Waste Sorter provides the Waste Vendor with Waste Log Information and in return obtains a Waste Pickup Date.
Provide Safe And Clean Space/
logWastePickupDate		 The Waste Sorter uses the system to create a Waste Pickup Request (creating a Waste Pickup Request ID and a Waste Pickup Request Date) and they input the Waste Pickup Date provided to them by the Waste Vendor.
Waste Sorter/
confirmWastePickup		 The Waste Vendor arrives at the facility and provides their Waste Vendor Information. They collect the Waste and provide the Waste Sorter with a Waste Vendor Receipt.
Provide Safe And Clean Space/
logWastePickupInformation		 The Waste Sorter uses the system search for the Waste Pickup Request by entering a Waste Pickup Request ID or Waste Pickup Request Date. They input the Waste Pickup Date Actual value and attach a scanned copy of the Waste Vendor Receipt.
