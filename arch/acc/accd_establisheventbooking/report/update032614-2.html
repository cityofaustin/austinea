Manage Maintenance Plan
Use Case

Use Case Scope

The Maintenance Manager uses the system to determine trends, usage and end of life information related to convention center assets. The Maintenence manager uses the system to communicate with the Facility Manager for resource needs. A new asset is entered into the system, and the Maintenance Manager specifies life cycle information (cost, replacement projection, maintenance schedule, vendor information, contract information, location, space served, warranty information, specifications, RFID, etc. ) The Maintenance Supervisor accesses work orders and updates the Maintenance Plan. The System generates a purchase request based on Maintenance plan. The Materials Planner uses the system to review Maintenance Plans and requested new/replacement assets and creates purchase requests as necessary.
