Create Project Charter
Use Case

Use Case Scope

A request is entered into the system by the Chain of Command for a project, and the Project manager is sent a notification of a new project. The information on the system includes "Official project name", Budget, which funding (CIP, FDU, etc.) , Estimate for when it would need to be completed, stakeholders, location, description of project. The Project Manager fills out the Project Charter template, and then schedules an appointment with Stakeholders. With Stakeholders, the project Manager fills out: Risks, Scope, Assumptions/clarifications, sets expectations, Other stakeholders. The Stakeholders vet the Charter, and the Sponsor and the Stakeholders sign the Charter.
