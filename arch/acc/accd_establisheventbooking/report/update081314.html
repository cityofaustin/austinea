Evaluate Sustainability for Purchases
Use Case

Use Case Scope

The Purchase Requestor uses the system to search for a desired Item to determine its Sustainability attributes. The lookup can be performed by entering item information. If the Item is found, the system will indicate whether the Item is sustainable or provide Alternative Items of the same ilk that are sustainable. If the item is not found or the item is found and it is categorized as 'not sustainable' and the re-evaluation date is past, the Purchase Requestor uses the system to send a notification to the Sustainability Coordinator with a request to evaluate the item. The Sustainability Coordinator receives the notification and completes the evaluation of the item. The Sustainability Coordinator uses the system to add the item. If the item is deemed sustainable, it is categorized as 'Sustainable'. If the Item is not sustainable, it is categorized as 'Not Sustainable' and alternative items are entered. The system requires the Sustainability Coordinator to enter a re-evaluation date on which to be reminded to re-evaluate the item for potential changes in its Sustainability attributes. The Sustainability Coordinator uses the system to send a notification to the Purchase Requestor informing them of evaluation results.
