Reserve Inventory
Use Case

Use Case Scope

Any Employee accesses the System and the System displays a list of only those Inventory Items to which the Employee has access/certification to reserve and are reservable. The Employee chooses the Inventory Item they would like to reserve, specifies Dates and Times and Reason/Work Order/Project/Business Use. The System communicates with the KeyWatcher system to make the Specified Reservation. The System sends email to the Key Master and the Maintenance Supervisor, informing them of the reservation. Note: The System should offer a Report of Scheduled Reservations for a specified Date Range.
