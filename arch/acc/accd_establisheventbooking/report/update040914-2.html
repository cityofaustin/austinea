Change Fleet Access Status
Use Case

Use Case Scope

The Security Staff receives notification that indicates access to Fleet Vehicles is now prohibited or allowed (DUI, Loss of License, Exceed License Points, Certification Expiration, etc). The Safety Security Staff accesses the System and records the Salient Information (Offense, point count, date, time, Name, etc). The System sends email to the Maintenance Supervisor indicating Change. Additionally, the System sends a message to the Key Master, notifying them of the change. Lastly, the System communicates with KeyWatcher to affect the change necessary (limit or release access to specific Key Sets). If any employee's Certification is due to expire, the System will send notification to the Employee and their Supervisor, informing them of pending expiration.
