Provide Project Online Admin Assistance
Use Case Model

Scope: The User fills out a helpdesk ticket and submits it. The Help Desk assigns the ticket to the Project Online Administrator. The Project Online Administrator contacts the User to get more details and then searches online resources like the knowledge base and Google to determine if a fix is known for the issue. If a solution is found, the Project Online Administrator implements the solution and closes the help desk ticket. If a solution is not found, the Project Online Administrator contacts the Vendor. The Vendor provides a solution to the Project Online Administrator, who then implements the solution and closes the Help Desk Ticket. If the Knowledge Base has no entry for the issue (or has an entry that requires modification), the Project Online Administrator contacts the Knowledge Base Single Point of Contact (SPOC) and requests the process be added or updated in the Knowledge Base repository. The Knowledge Base SPOC creates or updates the entry for the new issue.
