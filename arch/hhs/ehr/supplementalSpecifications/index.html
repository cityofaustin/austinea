Enterprise Architecture - aligning IT services with business needs


Table of EHR Supplemental Specifications

TypeDescription
Provide Fax Services The system provides a fax service - the service allows for the receipt of electronic fax documents entered into a queue. A system queue operator can route received faxes to designated parties via email or directly within the system. The system provides a means to attach received fax documents to designated health records. The system fax service provides for outgoing fax for scanned documents or directly from the system. The system provides shortcuts to fax address book items to search and retrieve frequently used fax numbers.
Provide Open Data Access The system provides access to all data records to include specialized record modifications or new entry forms created by the Clinic Staff. The data must be accessible to create reportable data-marts and data warehouses as needed to accomplish Health and Human Services (HHS) reporting needs. The system must provide open standard Web interface protocols in order for HHS to synchronize application information using internal enterprise service bus.
